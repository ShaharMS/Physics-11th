$\vec{A} = (a_x, a_y)\\$, $\vec{A} = (A_{length}, \theta_{degrees})\\$
$$
|\vec{A}|_{length} = \sqrt{a_x^2 + a_y^2}\\
\theta_{degrees} = tan^{-1}(\frac{a_y}{a_x})\\
a_x = |\vec{A}| * cos(\theta)\\
a_y = |\vec{A}| * sin(\theta)\\
$$

$Mathematical$ $Operations$
$$
\vec{A} + \vec{B} = (a_x + b_x, a_y + b_y)\\
\vec{A} + \vec{B} + \vec{C} + ... = (a_x + b_x + c_x + ..., a_y + b_y + c_y + ...)\\
\vec{A} - \vec{B} = (a_x - b_x, a_y - b_y)\\
\vec{A} + \vec{B} + \vec{C} + ... = (a_x + b_x + c_x + ..., a_y + b_y + c_y + ...)\\

$$
